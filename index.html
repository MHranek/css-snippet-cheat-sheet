Making Variables

How to structure the code for ccs variables. 

:root {
    --varname: content;
}

Linear Gradients

Creates a background linear gradient from top to bottom. 

.linear-gradient-background {
background-image: linear-gradient(
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
);
}

Use transition and box shadows to glow on hover.

border-radius: 8px;

Media Query

Primarily used for formatting between devices. 

@media screen and (max-width: 992px){
    .card {
        flex: 1 1 40%;
    }
}

Text-Wrap With pre

For things like sharing blocks of code, when the 'pre' tag is used this property can be leveraged to still allow text-wrapping. 

pre {
    white-space: pre-wrap;
}

CSS Grid Layout

Build a 12-column layout using CSS Grid 

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
}
